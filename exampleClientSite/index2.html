Open A/B

An easy-to-use, open-source solution for A/B testing
Get Started


1

Create Alternate Page

Start by making a copy of the page you want to A/B test, or build a completely different alternate version. The difference can be as slight as a change of color - or an entirely redesigned splash page. As long as both versions have the same target link, Open A/B can help you improve CTR.

2

Set Up Test

When both versions of a page are online, you are ready to set up a new test on Open A/B. From the dashboard, create a new test, entering your original page as the url. On the next screen, select the target link or button you want users to click. Then enter the url of the alternate version, and select the equivalent target link on this page.

3

Add Code Snippet

After you create a test, Open A/B will create a code snippet for you to put on both versions of the page you want to test. When you include this code, Open A/B will redirect 50% of your users to the alternate page and track views and conversions on both pages. The resulting statistics are displayed on the dashboard.
